Revolutionizing Personal Fitness

This project combines computer vision, machine learning, and cloud technology to create a smart fitness companion. The system provides real-time exercise form correction and personalized workout plans, all tracked and managed through a seamless mobile and wearable experience.

On-Device Intelligence

The core of the real-time feedback loop runs directly on the user's device for maximum speed and privacy. The process involves capturing video, running a sophisticated pose estimation model, and applying custom exercise logic to analyze the user's form instantly.

MediaPipe Pose Landmarker

A TFLite model that processes video frames to detect 33 key body landmarks.

Exercise Logic Module

Calculates joint angles from landmark data and compares them against correct form models.

Feedback Generator

Triggers audio cues and visual overlays based on the form analysis.

Cloud Data Composition

All user data is securely stored and managed in Google's Cloud Firestore. This central database enables progress tracking, workout history, and synchronization across devices. The data is structured to support the app's core features.
